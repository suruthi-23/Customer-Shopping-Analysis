# Customer Shopping Behavior Analysis

## Tools Used
- SQL (PostgreSQL)
- Python (Pandas, NumPy)
- Power BI

## Overview
Analyzed 3,900 customer purchase records to identify spending patterns, customer segments, and product preferences for data-driven decision-making.

## Key Insights
- Identified high-value customer segments driving overall revenue  
- Discovered key spending patterns and purchase trends  
- Analyzed top-performing products and customer preferences  
- Evaluated subscription and discount behavior
